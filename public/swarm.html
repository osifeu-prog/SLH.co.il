SLH Swarm — רשת פיזית מבוזרת

מכשירי ESP32 → טלפון Relay → Anchor Server. תקשורת mesh מבוזרת עם חתימות קריפטוגרפיות, ארנק כשר, proof-of-presence, ו-DePIN-lite עבור כלכלת SLH.

פרויקט תכנון — Phase 2/3 ב-Roadmap של SLH. Hardware POC טרם הוזמן. Blueprint טכני מלא זמין ב-ops/SWARM_V1_BLUEPRINT_20260424.md.

ארכיטקטורה · 3 שכבות

Layer 1

Device · ESP32

חומרה זולה (₪50/יחידה) שמתקשרת דרך ESP-NOW או WiFi.

  • תקשורת ESP-NOW (250m)
  • חתימה ECDSA מקומית
  • Cache ב-SPIFFS/LittleFS
  • כפתור פיזי לאישור
Layer 2

Relay · Phone

אנדרואיד (Termux POC) = gateway לאינטרנט.

  • BLE GATT / WiFi AP
  • WebSocket ל-Anchor
  • Buffer offline (SQLite)
  • חיבור לבוט טלגרם
Layer 3

Anchor · SLH API

אותו FastAPI על Railway — לא שרת נפרד.

  • Registry של devices
  • Hash-chain audit
  • Broadcast commands
  • Bridge ל-BSC/TON

זרימת מסר · Edge-First

ESP Device  ══ESP-NOW══>  ESP Device  ══BLE/WiFi══>  Phone Relay  ══WSS══>  SLH Anchor
     │                                                              │                        │
     ▼                                                              ▼                        ▼
  sign(ev)                                                   buffer + queue            events table
                                                                                      hash-chain audit
                                                                                      BSC bridge

Decision ladder:
  1. ESP decides alone if possible
  2. Phone intervenes if needed
  3. Anchor only if must

פרטי פרוטוקול SLH-MESH v1 + Protocol JSON format זמינים ב-blueprint הפנימי.
