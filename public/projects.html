מתעדכן אוטומטית לפי הפרויקטים שהוספת. תעתיק והדבק לסשן AI חדש.

שלבים הבאים — להתחבר טוב יותר

  1. API endpoint לפרויקטים — להוסיף ל-main.py: POST /api/projects/register + GET /api/projects. הפרויקטים יישמרו ב-DB ולא רק ב-localStorage.
  2. בוט integration — להוסיף לבוט פקודות /projects, /project add <name>, /scan <name>. קוד מוכן ב-TELEGRAM_CONTROL_GUIDE.
  3. GitHub Webhook — אוטומטית לרשום פרויקט חדש בכל git push. צריך endpoint שמקבל webhook secret.
  4. Drag-Drop → API — כרגע ה-uploads ב-localStorage. שדרוג: POST /api/uploads שמקבל blob, שומר ב-S3/Cloudflare R2, ושולח התראה לבוט.
  5. WebSocket live updates — כשסוכן מעדכן SYSTEM_ALIGNMENT, הדף הזה יראה זאת live ללא רענון.
  6. Per-project health checks — cron שמריץ curl לכל endpoint של כל פרויקט, מציג ירוק/אדום בדף.
  7. Agent claim board — חלק מהדף הזה: רשימת tasks פתוחים שסוכנים יכולים "לתבוע". מתנקה אוטומטית כשסוכן מסיים.
  8. Telegram Mini App — להגיע לדף הזה דרך Mini App מתוך @SLH_Claude_bot. Settings → Configure Mini App → URL: https://slh-nft.com/projects.html
