מה מפעיל ZUZ marking?

1. דיווחי קהילה

3 דיווחים עצמאיים = +10 ZUZ אוטומטי. דיווח כוזב = עונש על המדווח.

2. זיהוי אוטומטי

האלגוריתם מנטר: DMs המוניים, בקשות seed phrase, ניסיונות phishing.

3. OFAC Screening

כל ארנק עובר סינון מול Chainabuse + רשימות OFAC. ארנק חשוד = ZUZ מיידי.

4. On-chain Activity

קשר לארנקים ידועים כהונאה מטריגר סריקה דרך BSCScan API + Chainabuse API.
