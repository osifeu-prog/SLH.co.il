# SLH.co.il  מערכת ניהול ROI, בוט טלגרם וניטור

## 🚀 תכונות עיקריות
- בוט טלגרם לניהול מערכת (`/status`, `/users`, `/send_alert`, `/feedback`, `/request_admin`)
- Frontend סטטי (`www.slh.co.il`)
- לוח ניטור (`monitor.slh.co.il`)
- PostgreSQL ל‑ROI, feedback, משתמשים, אדמינים

## 🔧 טכנולוגיות
- Python (Telegram Bot)
- PostgreSQL
- Redis
- Next.js (עתידי)
- Railway (hosting)

## 📁 מבנה
- `bot.py`  הבוט הראשי
- `docs/`  Frontend סטטי
- `monitor/`  לוח ניטור
- `requirements.txt`  תלויות Python
