## Technical Specs Quick Reference

**Colors:**
- Primary: Cyan (#00FFFF)
- Secondary: Magenta (#FF00FF)
- Premium: Purple (#8800FF)
- Background: Dark (#0A0A1A)

**Typography:**
- Headings: Roboto Mono Bold
- Body: Courier New
- Terminal: Inconsolata

**Grid & Spacing:**
- 8px base unit
- 4:6:8:12:16:24:32:48 scale

**Animations:**
- Duration: 0.3-2s
- Easing: ease-out
- Respect prefers-reduced-motion

**Compliance:**
- WCAG 2.1 AA
- Color contrast 4.5:1+
- Keyboard navigation
- Screen reader support
