API ENDPOINTS:
==============

GET  /api/threat/check-score
  Query threat level for wallet/phone/user
  Returns: Arkham score + community reports + combined threat

POST /api/threat/report-fraud
  Submit community fraud report
  Returns: report_id for tracking

POST /api/threat/verify-report
  Admin verify/dismiss report
  Returns: verification result, reputation update

GET  /api/threat/leaderboard
  Get top detectives ranked by accuracy
  Returns: Top 20 reporters with stats

GET  /api/threat/network
  Analyze connections between bad actors
  Returns: Fraud ring visualization data
