## Token Distribution Model

### Registration Reward: 0.1 SLH per user
```
User pays 44.4 ILS (or TON equivalent)
    -> Admin approves
    -> 0.1 SLH credited to user's wallet
    -> Referral commissions distributed from payment
```

## Token Utility

### What SLH buys:
1. **Access**: Premium features on slh-nft.com
2. **Governance**: Vote on ecosystem decisions (planned)
3. **Staking rewards**: Earn yield by staking SLH
4. **Bot access**: Unlock premium bot features
5. **Marketplace**: Pay for goods/services in community store
6. **Referral boost**: Higher SLH balance = higher referral tier

### Earning SLH:
1. **Registration bonus**: 0.1 SLH on signup
2. **Referral commissions**: From downstream registrations
3. **Staking yields**: APY on staked tokens
4. **Community rewards**: Active posting, helpful content (planned)
5. **NFTY Bot**: Pet care XP converts to SLH rewards (planned)
6. **Achievements**: Milestone badges earn SLH (planned)
