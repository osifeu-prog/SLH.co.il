## 🧑‍💼 הוראות לך (אוסיף) — איך להשתמש בטמפלט

### בבוקר:
1. פותח סשן עם כל סוכן (Claude / ChatGPT / Core Assistant).
2. מדביק את הבלוק למעלה כהודעה ראשונה. זהו.
3. הסוכן יקרא את הקבצים ויאמר לך את המצב.
4. אתה מאשר פעולה → הוא מבצע → חוזר עם תוצאה.

- אל תעתיק לי מונולוגים ארוכים ממך-סוכן-אחר. זה שורף את הטוקנים שלך.
- אל תסביר לכל סוכן מה עשית — יש SESSION_STATUS.md בשביל זה.
- אם 2 סוכנים מתווכחים — תסגור לאחד מהם את הסשן. רק אחד מוביל בכל רגע.
