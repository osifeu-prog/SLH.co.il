TECHNICAL SPECIFICATIONS
================================================================================

COLOR PALETTE:
- Primary:   Cyan (#00FFFF) — Life, action, primary UI
- Secondary: Magenta (#FF00FF) — Power, energy, secondary UI
- Premium:   Purple (#8800FF) — Rarity, exclusive features
- Background: Dark (#0A0A1A) — Main background, eye-friendly
- Text:      White (#FFFFFF) — Clarity, readability
- Success:   Green (#00FF00) — Profit, positive outcomes
- Error:     Red (#FF0000) — Danger, negative outcomes
- Warning:   Yellow (#FFFF00) — Caution, attention needed

TYPOGRAPHY:
- Headings: Roboto Mono Bold (24px+, UPPERCASE)
- Body: Courier New (14px, 1.6 line-height)
- Small: Courier New (12px for secondary info)
- Terminal: Inconsolata (12px, cyan on dark)

DESIGN SYSTEM:
- 8px grid system
- 0.3-2s animation durations
- ease-out easing (natural feel)
- No pure black (#0A0A1A instead)
- No gradients (too soft)
- WCAG 2.1 AA compliance
- Respects prefers-reduced-motion
- Keyboard navigation support
