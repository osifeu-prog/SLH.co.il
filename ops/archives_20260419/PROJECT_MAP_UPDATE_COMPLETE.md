## 📝 DATABASE QUERY (for real-time updates)

If you want project-map.html to auto-update:

```sql
-- Get page completion stats
SELECT 
  COUNT(*) as total_pages,
  SUM(CASE WHEN score = 5 THEN 1 ELSE 0 END) as complete_count,
  SUM(CASE WHEN score >= 3 AND score < 5 THEN 1 ELSE 0 END) as in_progress_count,
  AVG(score) as average_score,
  SUM(CASE WHEN has_theme THEN 1 ELSE 0 END) as theme_count,
  SUM(CASE WHEN has_i18n THEN 1 ELSE 0 END) as i18n_count,
  SUM(CASE WHEN has_ai THEN 1 ELSE 0 END) as ai_count
FROM pages
WHERE deployed = true;
```
