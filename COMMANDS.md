# 🤖 SLH Claude Bot - מדריך פקודות

## 🚀 התחלה מהירה
- `/start` - הודעת פתיחה וקמפיין
- `/help` - רשימת כל הפקודות
- `/register` - הרשמה לעדכונים (חובה לכל תומך!)

## 💰 תרומה וקמפיין
- `/donate` - הצגת כתובת TON ורמות תמיכה
- `/status` - סטטוס פרויקט (Online/Active)

## 🎮 מעורבות ונקודות
- `/checkin` - צ'ק-אין יומי (+5 נק', streak)
- `/points` - הצגת הנקודות שלי
- `/leaderboard` - טבלת מובילים (טופ 5)
- `/daily` - משימות יומיות

## 👑 אדמין (רק למנהל)
- `/users` - רשימת כל המשתמשים הרשומים
- `/broadcast <הודעה>` - שליחת הודעה לכולם
- `/backup` - גיבוי נתונים בענן

## 🔧 טכני
- `/myid` - הצגת ה-ID שלי (לזיהוי)
- `/dashboard` - לינק לדשבורד
- `/crowdfunding` - קישור לעמוד הקמפיין
